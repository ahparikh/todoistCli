A simple cli client for [todoist](http://todoist.com/)

Install:

    sudo make install

~/.todoistCliCfg example:

    {
        //Your todoist API
        "api_token": "xxx",
        //The -i option default project id
        "project_id": 123 (must larger than 4), 
        //Alisa project id
        "alias": {
            "test(can't be number)":1350142
        }
    }
